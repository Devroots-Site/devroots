In den Bereich geht es um einen Restservice zu erstellen.

Die Technologien die dafür verwendet werden sind FastAPI und Uvicorn. 
FastAPI ist ein modernes Webframework für Python, das speziell für den schnellen Aufbau von APIs entwickelt wurde. 
Es basiert auf Starlette (einem asynchronen Webframework) und Pydantic (für Datenvalidierung) und bietet eine intuitive und performante Lösung für API-Entwicklung.
Uvicorn ist ein leichter und schneller ASGI-Server (Asynchronous Server Gateway Interface). Er dient als Server, der FastAPI-Apps ausführt und HTTP-/WebSocket-Anfragen verarbeitet.

Zur Installation schaue bitte im Bereich [FastAPI & Uvicorn](fastapiUndUvicorn.rst) nach.
