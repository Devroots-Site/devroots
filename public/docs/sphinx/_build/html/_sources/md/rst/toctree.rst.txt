RST Dateien
=================================
RST Dateien sind Dateien, die in der ReST (reStructuredText) Sprache geschrieben sind. ReST ist eine Markup-Sprache, die von Sphinx verwendet wird, um Dokumentationen zu schreiben. Es ist einfach zu lesen und zu schreiben und bietet viele Funktionen, die das Schreiben von Dokumentationen erleichtern.
